# CoverAI v3.0 — AI Генератор обложек для музыки

Профессиональный генератор обложек для музыкальных треков с AI-генерацией. Ориентирован на независимых артистов и новичков в дистрибуции музыки.

## Функции

- **AI-генерация**: 3-6 вариантов обложки по текстовому описанию
- **Квадратные форматы**: 1400×1400 — 6000×6000 px
- **8 стилей**: Minimal, Pop, Dark, Electronic, Indie, Rock, AI/Futuristic, Ethnic
- **Редактирование текста**: цвет, размер, позиция (свободное перемещение)
- **Экспорт**: JPG/PNG, RGB, 300 DPI
- **Референс-изображение**: загрузка для стилистического参考

## Быстрый старт

```bash
npm install
npm run dev
```

## Деплой на Vercel

```bash
npm i -g vercel
vercel login
vercel --prod
```

## AI API

По умолчанию используется демо-режим (mock AI). Для реальной генерации подключите:
- **Puter.js** — бесплатный фронтенд API (рекомендуется)
- **Replicate** — через API ключ
- **fal.ai** — через API ключ

Настройте API ключ в настройках приложения.
